# Features

| Feature | Description | Doc |
| ------- | ----------- | --- |
| module-installation | Resolve dependencies and install selected modules: packages (brew/apt), install.sh, commands, symlinks. | [module-installation](module-installation.md) |
| module-authoring | Create and edit modules via interactive forms, with built-in starter templates. | [module-authoring](module-authoring.md) |
| dotfile-management | Add dotfile mappings and import existing files/dirs (copy → remove → symlink → update config). | [dotfile-management](dotfile-management.md) |
| export-mode | Install only the dotfile symlinks, skipping packages and commands. | [export-mode](export-mode.md) |
